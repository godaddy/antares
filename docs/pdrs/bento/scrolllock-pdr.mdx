# ScrollLock Primitive PDR

## Purpose

The ScrollLock primitive provides a Bento-wrapped interface around React Aria's `usePreventScroll` hook, focusing primarily on API consistency and data attributes for debugging purposes.

**Research Finding**: React Aria's `usePreventScroll` includes comprehensive scroll prevention and layout stability features. According to the official documentation: *"Prevents scrolling on the document body on mount, and restores it on unmount. Also ensures that content does not shift due to the scrollbars disappearing."*

This research indicates that scrollbar compensation is already handled by the underlying React Aria implementation.

### Actual Unique Attributes

Given React Aria's comprehensive implementation, the ScrollLock primitive provides:
- **Data Attributes**: Adds debugging and styling hooks via `data-scroll-locked`
- **Environment Integration**: Uses Bento's Box context for consistent document/window access
- **TypeScript Types**: Provides Bento-aligned type definitions
- **Consistent API**: Follows Bento's hook patterns for familiarity
- **Optional Enhancements**: Additional configuration options if needed

## Primitive Composition

The ScrollLock primitive is built using:

1. **React Aria Foundation**:
   - `usePreventScroll` - Comprehensive scroll prevention including:
     - Touch events, iOS Safari issues, and proper cleanup
     - **Built-in scrollbar compensation** (prevents layout shift)
     - Keyboard scroll prevention and scroll chain management
     - Cross-browser normalization

2. **Minimal Bento Enhancements**:
   - Data attributes using `@bento/use-data-attributes` for debugging
   - Consistent TypeScript types with Bento patterns
   - API alignment with other Bento hooks

Given React Aria's built-in scrollbar compensation and layout shift prevention, this primitive serves as a **thin wrapper** that adds Bento-specific conventions while leveraging the underlying functionality.

- iOS Safari rubber-band scrolling prevention
- Proper cleanup and restoration
- Support for both document and element-level scroll locking

### Implementation Details

```tsx
import { usePreventScroll } from '@react-aria/overlays';
import { useDataAttributes } from '@bento/use-data-attributes';
import { useLayoutEffect } from 'react';

export function useScrollLock(options: ScrollLockOptions = {}) {
  const { isDisabled = false, ...reactAriaOptions } = options;

  // React Aria provides comprehensive scroll prevention
  usePreventScroll({
    isDisabled,
    ...reactAriaOptions
  });

  // Generate data attributes for debugging
  const dataAttributes = useDataAttributes({
    scrollLocked: !isDisabled
  });

  // Apply data attributes to document body
  useLayoutEffect(() => {
    const entries = Object.entries(dataAttributes);

    entries.forEach(([key, value]) => {
      if (value) {
        document.body.setAttribute(key, value);
      }
    });

    return () => {
      entries.forEach(([key]) => {
        document.body.removeAttribute(key);
      });
    };
  }, [dataAttributes]);
};
```

**Implementation Note**: React Aria's `usePreventScroll` provides the following features:
- Scrollbar width calculation and compensation
- Layout shift prevention
- Cross-browser scroll prevention
- Touch event handling
- iOS Safari fixes
- Proper cleanup and restoration

## Competitive Research

- **React Aria `usePreventScroll`**: Comprehensive solution with scrollbar compensation, mobile support, and accessibility
- **Radix UI `RemoveScroll`**: Good ref counting but less iOS support
- **Headless UI**: Built into Dialog, not available as standalone
- **body-scroll-lock`: Popular but abandoned, has memory leaks
- **react-remove-scroll`: Comprehensive but heavy dependency

**Key Finding**: React Aria's `usePreventScroll` is already the best-in-class solution. Our wrapper adds minimal value beyond API consistency and debugging attributes.

## Code Examples

### Basic Integration with React Aria Modal

```tsx
import { useModal, useOverlay, useDialog } from '@react-aria/overlays';
import { useScrollLock } from '@bento/use-scroll-lock';
import { withSlots } from '@bento/slots';

export const Modal = withSlots('BentoModal', function Modal(props) {
  const { isOpen, onClose, children } = props;
  const ref = useRef<HTMLDivElement>(null);

  // React Aria hooks for modal behavior
  const { overlayProps, underlayProps } = useOverlay(
    { isOpen, onClose, isDismissable: true },
    ref
  );
  const { modalProps } = useModal();
  const { dialogProps } = useDialog({}, ref);

  // Bento scroll lock (thin wrapper around React Aria's usePreventScroll)
  useScrollLock({
    isDisabled: !isOpen
  });

  if (!isOpen) return null;

  return (
    <div {...underlayProps} className="underlay">
      <div
        {...overlayProps}
        {...dialogProps}
        {...modalProps}
        ref={ref}
        slot="modal"
      >
        {children}
      </div>
    </div>
  );
});
```
