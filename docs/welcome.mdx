# Bento

**Bento** is a headless, accessible React component library built on [React Aria](https://react-spectrum.adobe.com/react-aria/). It provides the foundational primitives for building design systems.

## Vision

Bento aims to be the **invisible foundation** that powers your design system. We believe:

- **Accessibility is not optional** — Every component is built with WCAG compliance from day one
- **Flexibility over opinions** — Primitives should adapt to your design, not the other way around
- **Composition over configuration** — Build complex UIs by combining simple, focused primitives
- **Developer experience matters** — Clear APIs, excellent TypeScript support, and comprehensive documentation

You provide the styles, we handle the hard parts.
