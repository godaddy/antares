TooltipDismissStrip

Internal visx XYChart helper that closes the tooltip when the pointer enters a leading-edge strip over scroll-synced axis chrome

Internal Component

This component is internal-only and not exported from the package. It is designed for use within chart components.

Props

The TooltipDismissStrip component accepts the following props:

Width in px pinned to the inline-start viewport edge (visual left in LTR, visual right in RTL); match sticky axis chrome that overlaps the plot after horizontal scroll.
