Popover

The Popover component is an overlay element positioned relative to a trigger element.

Examples

Basic Usage

Use it with <PopoverTrigger>, which wraps both the popover content and an interactive trigger element (for example, a Button).

Custom Anchor

To position a popover relative to a different element than its trigger, use the triggerRef and isOpen props instead of <PopoverTrigger>. onOpenChange is called when the user opens or closes the popover.

Popover will be positioned relative to me

Close Button and Header

To show a close button in the popover header, use the showCloseButton prop.

You can also use the header prop to render content alongside the close button. This is useful for displaying a title or other header content without overlapping the close button.

Best Practices

Popover width is determined by its content, and it’s recommended to keep it under 400px.
