## Features

- Slides in from any screen edge (top, bottom, left, right)
- `overlay` elevation panel over a dimmed backdrop, matching `Modal`
- Automatic RTL flipping for left/right placement
- Focus trapping, backdrop dismiss, and scroll lock via RAC Modal
- CSS transitions with `prefers-reduced-motion` support

```bash
npm install --save @godaddy/antares
```

## Accessibility

### Keyboard

| Key | Action |
| --- | --- |
| `Escape` | Close the drawer (unless `isKeyboardDismissDisabled`) |
| `Tab` | Move focus within the drawer (focus is trapped) |

### ARIA

- `role="dialog"` with `aria-modal="true"` on the drawer panel (via RAC Modal)
- Dialog accessible name via `title` (renders a heading) or `aria-label` (for titleless drawers)
- `aria-label="Close"` on the built-in close button

## Best Practices

- Always provide a `title` or `aria-label` so the dialog has an accessible name.
- Use `isDismissable` to allow closing via backdrop click.
- Don't nest drawers.
