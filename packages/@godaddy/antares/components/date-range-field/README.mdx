## Features

- **Calendar-first**: clicking the field opens a `RangeCalendar` popover; the selected range is shown as a localized value (e.g. `12 â€“ 20 Aug 2022`)
- **Optional keyboard entry**: set `allowsKeyboardInput` for editable start/end segments with an icon-only calendar trigger
- **Localized display**: the range is formatted with `useDateFormatter`'s `formatRange`, which collapses shared month/year per locale; override with `formatOptions`
- **Bounds and availability**: `minValue` / `maxValue` and `isDateUnavailable` pass straight through to the inputs and calendar
- **React Aria integration**: built on React Aria `DateRangePicker` for accessibility, forms, and locale handling

## Installation

```bash
npm install --save @godaddy/antares
```

## Working with dates

`DateRangeField` is typed for `CalendarDate` from `@internationalized/date`. See the
[`DateField`](/docs/components-datefield--overview) docs for the canonical "Working with dates"
guide â€” installation, `<I18nProvider>`, locale handling, and common patterns all apply here. A
range value is `{ start, end }` of `CalendarDate`:

```tsx
import { parseDate } from '@internationalized/date';
import { DateRangeField } from '@godaddy/antares';

<DateRangeField
  label="Trip dates"
  defaultValue={{ start: parseDate('2024-03-15'), end: parseDate('2024-03-20') }}
/>;
```
