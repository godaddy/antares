## Accessibility

### ARIA Support

- `role="progressbar"` on the container
- `aria-valuenow` reflects the current value
- `aria-valuemin` and `aria-valuemax` define the range (default `0`–`100`)
- `aria-valuetext` provides a formatted string (e.g. `"60%"`)
- `aria-labelledby` associates the visible label when the `label` prop is provided
