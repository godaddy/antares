## Accessibility

### ARIA Support

- `role="progressbar"` on the container
- `aria-valuenow` reflects the current value (clamped to 0–100)
- `aria-valuemin` and `aria-valuemax` define the range (default `0`–`100`)
- `aria-valuetext` provides a formatted string (e.g. `"60%"`) via `Intl.NumberFormat`; the same string is rendered inside the circle. Override the text with the `aria-valuetext` prop, or control formatting with `formatOptions`
- `aria-labelledby` associates the visible label when the `label` prop is provided
- `aria-describedby` wires the `helperText` to the progressbar for screen readers
- The decorative SVG and output text are hidden via `aria-hidden="true"`
