# @godaddy/generate-cdn-url

## 0.0.0

### Initial Release

- Initial implementation of CDN URL generation library
- Support for package-based URL generation (`{cdn}/{packageName}/{version}/{assetPath}`)
- Support for flexible path segments mode
- Special handling for versions.json endpoint (`version: 'all'`, `'list'`, or `'json'`)
- Hostname replacement support for environment switching
- Zero dependencies
- Full TypeScript support with type definitions
- 100% test coverage
