***REMOVED***SVG Sprite Concepts

SVG sprites are like a collection of icons stored in a single, hidden container. 
Instead of loading each icon separately, we keep them all in one place and 
reference them when needed. This makes our app faster and easier to maintain.

Think of it like a sticker book: instead of having loose stickers everywhere, 
they're all organized in one book, and you can use them whenever you need them.

#***REMOVED***Using useSVGSprite

The `useSVGSprite` hook simplifies working with SVG sprites in React applications. 
It handles all the sprite sheet management for us, making it easy to use icons

#***REMOVED***Sprite Sheet Structure

Let's look at how the sprite sheet and icon usage work together. First, we 
define our icons in the sprite sheet:

```html
<svg id="bento-svg-spritesheet" style="display: none">
  <symbol id="bento-svg-spritesheet-check" viewBox="0 0 24 24">
    <path d="M5 13l4 4L19 7" stroke="currentColor" strokeWidth="2" fill="none" />
  </symbol>
  </symbol>
</svg>
```

Then, whenever we need to use an icon, we reference it from the sprite sheet 
using its ID. For example, to use the check icon:

```html
<svg viewBox="0 0 24 24" width="24" height="24">
  <use xlinkHref="#bento-svg-spritesheet-check" fill="currentColor"/>
</svg>
```

The connection between the sprite sheet and icon usage is simple. Each icon in 
the sprite sheet has a unique ID, like `bento-svg-spritesheet-check`. When we 
want to use an icon, we reference it using that same ID. The browser then finds 
the icon in the sprite sheet and displays it where we want it.
