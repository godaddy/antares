# Drawer

The `@bento/drawer` package exports the **Drawer** component, which creates an expandable panel that can be used for sidebar navigation, bottom sheets, and disclosures.

## Installation

```bash
npm install --save @bento/drawer
```

## Accessibility

The Drawer component is a fully controlled component that requires you to manage its state:

- **ARIA attributes**: You should set `aria-expanded` based on your state management
- **Role support**: Can be configured with `role="dialog"` for modal-like behavior or `role="region"` for non-modal content
- **State management**: Use `useState` or your preferred state management solution to control `isExpanded`

## React Aria Integration

The Drawer component can be used with React Aria hooks for advanced behaviors:

- **useOverlay**: Can be used in your components to handle overlay behavior (dismissal, keyboard interactions) when the drawer is portaled
- See the dismissal example for a complete implementation using React Aria hooks
