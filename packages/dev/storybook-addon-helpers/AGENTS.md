# @bento/storybook-addon-helpers

Guide for working with the Storybook addon helpers package.

## Build/Test

Run tasks through Nx, scoped to `@bento/storybook-addon-helpers` (prefix with the package manager, e.g. `npm exec nx`; don't run these on other workspaces):

```sh
npm exec nx run @bento/storybook-addon-helpers:typecheck
npm exec nx run @bento/storybook-addon-helpers:lint
npm exec nx run @bento/storybook-addon-helpers:build
npm exec nx run @bento/storybook-addon-helpers:test
```
